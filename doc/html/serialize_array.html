#define serialize_array

(java_type_name, cpp_type,
  serial_version_uid)

Define the serializable array.

Documentation

Define the serializable array. The Java builtin types are supported. The macro can be used for user specified classes that act as C++ builtin types.

Example (file java_lang.cpp):

. serialize_array("I", int, 0x4DBA602676EAB2A5);

Parameters:
java_type_name - - a Java type name as a string of the array elements.
cpp_type - - the C++ type the array elements.
serial_version_uid - - a serial version uid number as specified in java implementation of the class 'private static final long serialVersionUID' variable should be specified for any serializable class. More then one class can have the same serialVersionUID. Java 2 SDK provides 'serialver' RMI tool that returns a class serialVersionUID.
