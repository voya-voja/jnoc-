Documentation

A Java object parser encapsulates the parsing of Java object stream. It provides the access to a object symbol sent using Java object stream, if the stream has been parsed without error(s).

enum State
The states of a JavaObjectParser when reading the object stream:

k_ReadBoolean, k_ReadByte, k_ReadChar, k_ReadShort, k_ReadInt, k_ReadLong, k_ReadFloat, k_ReadDouble, k_ReadString, k_ReturnByte
