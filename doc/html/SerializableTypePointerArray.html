Documentation

Specialization of Serializable class that provides means to define a C++ serializable array of pointers. The class uses inexum::VectorPtr provide the support of a Java array and garbage collection of contained objects.

SerializableTypePointerArray( const std::string& className )
Overloaded constructor creates a serializable wrapper for an array of pointers. The class name is expected to be in format '['<base element type>.

Parameters:
- className - the Java object stream array name.
