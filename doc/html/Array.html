void setName(const std::string& name)
Provide more descriptive array name and to capture the elements' type. The name should corresponde to java serialization format, e.g. '[B' or '[Ljava.lang.Long;'.

Parameters:
- name - a string name in Java serialization format

void addElements(int count)
Add specified number of elements of arrays type. It can be called only after setName, because it requires elements' type.

Parameters:
count - - the number of elemnets to be added
