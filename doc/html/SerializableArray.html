Documentation

Specialization of Serializable class that provides means to define a C++ serializable array. The class uses std::vector class to provide the support of a Java array.

SerializableArray(const std::string& className)
Overloaded constructor creates a serializable wrapper for an array. The class name is expected to be in format '['<element type>.

Parameters:
- className - the Java object stream array name.

JLong serialVersionUID() const
Specialize for diferent instantiation of SerializableArray.
