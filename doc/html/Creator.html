Documentation

A labeled factory method creates a product specialized from a base product using product's default constructor. Once Creator temlate is instantiated it will register it self into the factory instantiated for the same lable type and base product. The same instance of the Creator will be used to create instances of the product that would be returned by the factory as a pointer to a base product.

NOTE: Make an instance of this class in a cpp file, e.g. if MyBaseProductClass is a base class of ProductClass0, then:

template Creator< int, 0, MyBaseProductClass, ProductClass0>;
