JNOC++ library

'Java Network Objects in C++'

Made and maintained by iNexum Systems Inc. the JNOC++ system is a full lightweight network object communicaiton library for high speed inter language communication.

JNOC++ is released under the GNU Library General Public License and is available to download.

Where did JNOC++ come from?

JNOC++ stands for Java Network Object Communications framework. The program was written by Nebojsa Vojinovic for iNexum's MicroAc Cashless payment processing system, in response to a need for a library to provide simple connectivity between Java client and C++ server.

Is JNOC++ thread safe?

This version is not thread safe, or more then one thread can not use the same Object stream to write or to read. However, if the physical stream is thread safe then one thread can read, while the other writes.
